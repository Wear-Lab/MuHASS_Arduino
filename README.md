## Libraries needed 
From Library Manager:
* Adafruit APDS9960 Library
* Adafruit BluefruitLE nRF51
* Adafruit BMP280 Library
* Adafruit BusIO
* Adafruit LIS3MDL
* Adafruit LSMDS
* Adafruit SHT31 Library
* Adafruit Unified Sensor
* RTClib
* SparkFun MAX3010x Pulse and Proximity Sensor Library
